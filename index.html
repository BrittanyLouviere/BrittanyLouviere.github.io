Progress Bar App

In this application users can track their progress on any enumerable goal they set. The app will tell the user how many actions per day (or how many days per action) they need to do to stay on track and display their progress on a colorful bar. If part of the bar displays in red, the user is behind on their goal and will need to catch up. If it is green the user is ahead.


Languages and frameworks used in this project:

  • Vue.js
  • HTML
  • CSS
  • Javascript
